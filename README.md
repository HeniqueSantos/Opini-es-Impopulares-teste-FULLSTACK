# Opiniões Impopulares - Teste Fullstack

Projeto simples em **Node.js** com **Express** e **MySQL** para praticar backend e manipulação de dados.

---

## Funcionalidades

- Exibir opiniões.
- Curtir e descurtir opiniões.
- Backend simples rodando com Node + Express + MySQL.

---

## Pré-requisitos

- [Node.js](https://nodejs.org/) instalado.
- [MySQL](https://www.mysql.com/) instalado e rodando.

---

## Como rodar o projeto

1. Clone o repositório:
git clone https://github.com/HeniqueSantos/Opini-es-Impopulares-teste-FULLSTACK.git

Entre na pasta do projeto:
cd Opini-es-Impopulares-teste-FULLSTACK

Instale as dependências:
npm install

Configure o banco de dados MySQL:
Crie o banco.
Ajuste as credenciais no arquivo server.js (usuário, senha, nome do DB).

Rode o servidor:
node server.js

Abra no navegador:
http://localhost:3000
